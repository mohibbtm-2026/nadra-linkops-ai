# Public Source Notes for Quetta-Region Site Directory

This project includes a **public-source-based site directory** for Quetta-region NADRA offices/centres.

## What was found publicly
Publicly searchable NADRA pages list office/centre names, districts, addresses, and office timings for many Balochistan/Quetta-region locations.
Examples include:
- NRC RHO Quetta (Quetta-III) Morning/Evening Shift
- Mega Center Quetta Morning/Evening Shift
- NRC Pishin
- NRC Chaman
- NRC Qilla Abdullah
- NRC Loralai
- NRC Harnai
- NRC Ziarat
- NRC Zhob
- NRC Sherani
- NRC Qillah Saifullah
- NRC Khuzdar
- NRC Dalbandin
- NRC Noshki
- NRC Sibi

## What was NOT found publicly
Public NADRA pages do **not** reliably publish operational **DAU incharge names** for these sites in a way suitable for this network operations project.
Because of that, the app uses:
- public site names and location details where available, and
- manually editable incharge fields that the student/user can replace with real internal data if authorized.

## Public URLs referenced during research
- Main NADRA website: https://www.nadra.gov.pk/
- NADRA office/location page: https://www.nadra.gov.pk/nadra-locations/
- Balochistan office listing result surfaced by search: https://balochistansuccession.nadra.gov.pk/offices/
- NADRA eSahulat locations CSV: https://www.nadra.gov.pk/eSahulat/Esahulat_Locations.csv

## Academic integrity note
This approach avoids inventing official incharge names and keeps the assignment realistic, ethical, and defensible.
